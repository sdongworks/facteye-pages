팩트아이(AI) (FactEye)

개인정보처리방침

시행일: 2026년 6월 15일

팩트아이(AI)는 사용자의 개인정보를 수집하지 않습니다.
분석에 필요한 유튜브 URL만 서버로 전송되며, 결과는 24시간 이내 자동 삭제됩니다.

1. 수집하는 정보

팩트아이(AI) 앱은 다음 정보만 처리합니다.

  • 유튜브 영상 URL — 사용자가 직접 입력하거나 공유한 주소

이름, 이메일, 전화번호, 위치 등 개인 식별 정보는 일절 수집하지 않습니다.

2. 정보 이용 목적

전송된 URL은 아래 목적으로만 사용됩니다.

  • 유튜브 영상 자막 추출 및 내용 요약
  • 영상 내 언급 항목(인물·제품·기관·주장) 분석
  • 관련 네이버 검색 링크 제공

3. 제3자 서비스 이용

영상 분석 과정에서 아래 외부 서비스에 URL 또는 자막 텍스트가 전달됩니다.

  • Supadata — 유튜브 자막 추출 (supadata.ai)
  • OpenAI — 내용 요약 및 분석 (GPT-4o-mini) (개인정보처리방침)
  • Naver Search API — 관련 링크 검색 (개인정보처리방침)

각 서비스는 해당 업체의 개인정보처리방침을 따릅니다. 전달되는 정보는 영상 URL과 자막 텍스트뿐이며, 사용자의 신원 정보는 포함되지 않습니다.

4. 정보 보관 및 삭제

  • 분석 결과는 서버 메모리에 최대 24시간 캐시된 후 자동 삭제됩니다.
  • 별도 데이터베이스에 저장하지 않습니다.
  • 사용자 계정이 없으므로 별도 탈퇴 절차가 필요 없습니다.

5. 아동 개인정보 보호

팩트아이(AI)는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 방침 변경 안내

본 방침이 변경될 경우 이 페이지를 통해 공지하며, 시행일을 상단에 표시합니다.
